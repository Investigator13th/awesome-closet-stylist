# 👗 Awesome Closet Stylist | 私人衣橱管家

Maintain a persistent personal wardrobe, answer wardrobe questions, and recommend outfits from existing clothes with natural-language responses. 
通过自然语言交互，维护您的持久化个人衣橱，回答关于衣物的各种疑问，并为您提供定制化的穿搭方案推荐。

---

### ✨ Features | 功能亮点

- 🗄️ **Smart Wardrobe** | **智能衣橱**：轻松记录、更新和管理您的所有单品。
- 💃 **Outfit Planner** | **穿搭助手**：根据天气、场合和个人偏好，从现有衣物中精选推荐方案。
- 💬 **Natural Chat** | **自然交互**：像和真人管家聊天一样沟通您的穿搭需求。
- 📝 **Style Memory** | **风格记忆**：记录您的喜好与反馈，让推荐越用越懂你。
